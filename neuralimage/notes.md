### Новое

- Добавлено раздельное хранение путей для разных режимов работы:
  `обучение и распознавание`, `дообучение и распознавание`,
  `обучение`, `распознавание`.
- При переключении режима приложение восстанавливает свои пути и число эпох
  для выбранного сценария.

### Интерфейс

- Скрываются поля, которые не используются в текущем режиме.
- В режиме `Распознавание` скрывается блок `Обучающая выборка`.
- В режиме без обучения скрывается поле количества эпох.
- В режимах без загрузки внешней модели скрывается поле файла модели.

### Обновления

- Updater теперь поддерживает release notes в формате Markdown.
- Release notes можно передавать прямо в `version.json` или через отдельный
  файл `notes.md`.
- Добавлена поддержка каналов обновлений.
- В `version.json` можно указывать канал релиза, например `stable` или `beta`.
- В программе можно выбрать, с какого канала проверять и устанавливать
  обновления.

### Совместимость

- Для использования каналов обновлений рекомендуется новый формат
  `update_client.json` с секцией `channels`.
- Для release notes в `version.json` можно использовать поля
  `release_notes_markdown`, `release_notes_path`, `notes_markdown`,
  `notes_path`.
