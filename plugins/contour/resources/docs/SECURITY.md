# Security Policy

## Supported versions

Only the latest minor release line is supported with security fixes.

| Version | Supported |
|---------|-----------|
| 0.4.x   | Yes       |
| < 0.4   | No        |

## Reporting a vulnerability

Please do **not** open a public GitHub issue for security reports.

Instead, email the maintainer at **shustinskimark@gmail.com** with:

- A description of the issue.
- Steps to reproduce (including a proof of concept if available).
- The impact you anticipate.
- Any suggested mitigations.

You will receive an acknowledgement within **72 hours**. We aim to ship a fix
or a mitigation plan within **30 days** of the initial report, and will
coordinate public disclosure with you.
