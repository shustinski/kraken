# Karakal

Karakal - это Qt-плагин для Kraken, который помогает строить матрицу кадров, сравнивать результаты моделей и выбирать кадры для первичной разметки.

Проект работает вокруг двух основных сценариев:

- режим валидации, где строится и анализируется матрица кадров;
- режим менеджера, где та же матрица используется для отбора кадров на разметку.

## Что умеет Karakal

- строить матрицу по индексированным кадрам и сохранять её раскладку;
- рассчитывать метрики для сравнения моделей и оценки кадров;
- работать с confidence/uncertainty-видами для выходов моделей;
- открывать подробный просмотр кадра с тайловым и subpixel-видами;
- показывать manager mode для первичной выборки под разметку;
- окрашивать кластеры отдельным цветом и помечать рекомендованные кадры поверх матрицы;
- использовать diversity-отбор, чтобы выборка покрывала разные структуры, а не дублировала почти одинаковые кадры.

## Основные части интерфейса

- Источники: папки моделей, исходные кадры, разметка и папка экспорта.
- Параметры анализа: режим матрицы, режим оценки, геометрия, confidence-настройки.
- Матрица валидации: сетка кадров для просмотра и сравнения.
- Матрица менеджера: режим для отбора кадров на первичную разметку и контроля кластеров.
- Детали кадра: полный просмотр отдельного кадра, включая тайлы и subpixel-переходы.

## Режим менеджера

Режим менеджера нужен для того, чтобы понять, какие кадры стоит размечать первыми.

Сейчас он:

- считает приоритет первичной выборки;
- группирует кадры по структурному сходству;
- старается брать минимум кадров из каждого не доминирующего кластера;
- использует ту же матрицу, что и режим валидации.

## Типовой сценарий

1. Добавить папки с результатами моделей.
2. Указать папку исходных кадров, если нужна первичная выборка.
3. Построить матрицу.
4. Посчитать аналитические метрики.
5. Просмотреть кадры в режиме валидации.
6. Переключиться в режим менеджера и посмотреть рекомендованные кадры и кластеры.

## Запуск

Karakal предназначен для работы внутри Kraken как плагин.

Для локального запуска в разработке:

```bash
python -m karakal
```
